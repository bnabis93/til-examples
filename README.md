### Rules  
- 한국어로 정리 하려함

- 예를들어 backend/cors.md의 예제로써 backend/cors-example.py
- 남의 글을 내꺼인것처럼 쓰지 말자. 나의 언어로 바꾸어 정리한다. 최대한 핵심만 간결하게.
